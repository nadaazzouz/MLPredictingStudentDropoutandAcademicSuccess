## Project Description

This project implements a complete Machine Learning pipeline to predict whether a university student will drop out, remain enrolled, or graduate as early as the end of the first semester, allowing the university to intervene before it is too late.


## Dataset

| Attribute | Detail |
|---|---|
| **Name** | Predict Students' Dropout and Academic Success |
| **Source** | [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success) |
| **Origin** | Polytechnic Institute of Portalegre, Portugal (2021) |
| **Dimensions** | 4 424  students × 36 columns |
| **Features** | 35 academic, demographic, and socio-economic variables |
| **Target** | Categorical : `Dropout` / `Enrolled` / `Graduate` |

**Justification for choice:** Real dataset from a Portuguese university. Clear problem with strong social impact. Sufficient size (>500 rows). Diverse variables enabling rich feature engineering. Direct access via ucimlrepo ensures full reproducibility.

## Problem Statement

How to identify, at the end of Semester 1, students at risk of dropping out in order to enable early intervention by the institution

- Problem type: supervised multi-class classification  
- Target variable: Target (Dropout=0 / Enrolled=1 / Graduate=2)  
- Main metric: weighted F1-score, suitable for class imbalance

| Decision Tree | ~76% | ~75% | ~83% |
| Random Forest | ~86% | ~85% | ~94% |

| **XGBoost (optimisé)** | **~88%** | **~88%** | **~96%** |

### Top 5 Dropout Factors using SHAP
1. **Second semester grade** strongest predictive signal
2. **Approved units in second semester** academic progress
3. **First semester grade** early signal
4. **Tuition fees up to date**  financial factor
5. **Age at enrollment** adult student profile

### Business Recommendations
- Early warning system based on first semester performance
- Targeted financial support program for students with unpaid fees
- Dedicated support system for adult students older than 23
